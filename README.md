# Relationship NOTE

> 개인프로젝트

> 사람과의 긍정적인 인간관계를 만드는 웹 노트

## 사용기술

- 라이브러리 : React js
- 프론트앤드 프레임워크 : react-bootstrap, tailwind
- 백앤드 프레임워크 : Express js
- 언어 : javascript
- 데이터베이스 : Mysql
- 플랫폼 : AWS ec2, s3
